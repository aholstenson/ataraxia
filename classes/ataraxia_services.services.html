get

  • get<S>(id: string): null | (Service & S)
    • Defined in packages/services/src/Services.ts:230

    Get a service if it is available. Will return a proxy that can be used to invoke methods on the service regardless if it is remote or local:

    const instance = services.get('example:test');
if(instance) {
  await instance.doStuff();
}

    For TypeScript it is possible to scope it to a known interface:

    interface TestService {
  doStuff(): Promise<void>;
}

const instance = services.get<TestService>('example:test');
if(instance) {
  await instance.doStuff();
}

    Type parameters

    • S: object

    Parameters

    • id: string

      - the identifier of the service

    Returns null | (Service & S)

    Service instance or null if service is unavailable
